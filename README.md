# Project Setup

## Prerequisites

- [Docker](https://www.docker.com/) installed
- OpenAI API Key

## Getting Started

Follow these steps to run the project:

1. Open the `docker-compose.yml` file.
2. Set your OpenAI API key by updating the `OpenAI__ApiKey` environment variable.
3. Start the project by running:

   ```bash
   docker-compose up -d
Open your browser and go to http://localhost:4300.

Add a receipt (a sample receipt is available in the assets folder).

Notes
Make sure your OpenAI API Key is active and has sufficient usage limits.
